# Project: Alexa Grok

## Overview
An Alexa skill that interacts with xAI's Grok API for conversational AI.

## Objective
Migrate from OpenAI GPT API to xAI Grok API, specifically using the `grok-4-1-fast-non-reasoning` model.

## Technology Stack
- Alexa Skills Kit (ASK)
- AWS Lambda (Python)
- xAI Grok API (OpenAI-compatible)

## Constraints
- Alexa Lambda timeout (8 seconds)
- API responsiveness
- Secure API key management
